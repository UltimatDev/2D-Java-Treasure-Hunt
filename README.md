# 2D Java Treasure Hunt

A 2D adventure game built using Java to explore core Object-Oriented Programming concepts and game development fundamentals.

## 🚀 Overview

This project is a simple 2D treasure hunt game where the player navigates through a map, overcomes obstacles, and searches for hidden treasure. The game is designed to demonstrate how Java can be used to build interactive graphical applications while applying clean OOP principles.

The project focuses on:
- Game loop implementation
- Player movement & controls
- Collision handling
- Basic rendering using Java graphics
- Modular design using OOP

---

## 🧠 Concepts Applied

- Object-Oriented Programming (Encapsulation, Inheritance, Polymorphism)
- Event-driven programming
- Game state management
- Rendering & animation basics
- Input handling (keyboard controls)

---

## 🎮 Features

- Smooth player movement
- Interactive game environment
- Treasure discovery mechanics
- Modular and extendable code structure
- Basic 2D rendering system
